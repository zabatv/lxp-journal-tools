# LXP Journal Tools

Веб-приложение для автоматического заполнения DOCX-журналов данными из LXP.

## Возможности

- Авторизация через LXP (email/password)
- Выбор подорганизации, учебного периода, группы и дисциплины
- Автоматическая загрузка студентов и их оценок
- Заполнение DOCX-шаблона с плейсхолдерами:
  - `%g` — название группы
  - `%d` — название дисциплины
  - `%t` — ФИО преподавателя
  - `%n` — ФИО студента
  - `%q` — оценка студента
